Personal learning journal and proof-of-work for AI × Web3 School.

- WCB Course: https://web3career.build/zh/programs/AI-Web3-School

- WCB Agent API Docs: https://web3career.build/llms.txt
- Learning Agent Prompt: https://aiweb3.school/learning-agent.zh.txt

## Privacy

This repository is intended to be public. Do not store API keys, private keys, seed phrases, passwords, personal contact details, private meeting links, or any other sensitive information here.

## Structure

- `profile.md`: learner profile and operating preferences.
- `learning-plan.md`: current learning plan and weekly rhythm.
- `daily/`: daily notes and check-in records.
- `tasks/`: task breakdowns and proof-of-work notes.
- `experiments/`: code, prompts, demos, and technical experiments.
- `handbook-feedback/`: structured feedback for Handbook pages.
- `hackathon/`: hackathon ideas, prep, and project notes.
- `submissions/`: links and records for platform submissions.
- `templates/`: reusable note templates.

## Working Rules

1. Use the repo as a public proof-of-work workspace, not just a notebook.
2. Write daily notes before or during study, and append submission links after check-in.
3. Keep Handbook questions and feedback in `handbook-feedback/`.
4. Review `git status --short` before each commit.
