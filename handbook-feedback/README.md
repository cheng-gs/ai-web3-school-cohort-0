# Handbook Feedback Workflow

Use this folder for problems, confusion points, stale references, wording issues, or structural suggestions found while studying the Handbook.

## One Feedback Per File

Suggested filename:

`YYYY-MM-DD-short-topic.md`

## Required Fields

- Handbook page link
- Problem type
- What is unclear or outdated
- Suggested change
- Source from your study process

## Flow

1. Capture the issue while studying.
2. Write a file from the template in this folder.
3. Link the feedback file from the daily note if relevant.
4. Commit it with your learning notes after review.
