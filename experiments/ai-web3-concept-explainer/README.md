# AI × Web3 概念解释器

这是一个 Week 1 的最小可交互学习产物原型。

## 解决什么问题

AI 和 Web3 的初学者常常会遇到两个问题：

- 术语很多，但彼此之间的关系不清楚
- 听过定义后，仍然不知道怎么用自己的话解释，或下一步该做什么

这个原型用固定模板把一个概念拆成：

- 解释
- 例子
- 类比
- 误区或边界
- 下一步行动

## 如何使用

直接用浏览器打开 [index.html](./index.html)。

输入内容：

- 选择概念
- 选择自己的背景
- 选择输出风格
- 写下当前困惑

点击“生成解释草稿”后，会得到一份适合写进学习笔记或打卡草稿的内容。

## 说明

- 这是一个本地静态网页，不依赖后端和模型 API
- 初版结构和文案框架由 AI 辅助起草
- 概念边界、措辞和安全提醒由人工复核和改写
- 它适合学习辅助，不适合当作正式定义或链上操作依据
